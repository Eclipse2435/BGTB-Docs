Features

  • Earn tickets by playing minigames.
  • Enter giveaways using tickets.
  • Host giveaways as an admin.

Commands

/earn

Description: Play a minigame to earn tickets.

Usage: /earn

Channel: Only works in the designated ticket-earning channel.

/inventory

Description: Check your ticket inventory.

Usage: /inventory

/enter_giveaway

Description: Enter the giveaway using 1 ticket.

Usage: /enter_giveaway

Condition: Only works if a giveaway is active.

/host_giveaway

Description: Start a new giveaway.

Usage: /host_giveaway <duration>

Permissions: Admin only.

/end_giveaway

Description: End the current giveaway.

Usage: /end_giveaway

Permissions: Admin only.

/announce_winner

Description: Announce the winner of the giveaway.

Usage: /announce_winner <@user>

Permissions: Admin only.

Setup

Required Intents: Ensure Message Content Intent and Server Members Intent are enabled in the Discord Developer Portal.
